# Marine Engine Log Parser

## What I Need
A tool that helps me parse marine engine logs and identify critical issues quickly during my shift.

## Target Users
- Marine chief engineers on cargo ships
- Engine room operators

## Environment
- Noisy engine room
- Limited or no internet connection
- Need quick access during 12-hour shifts
- Often wearing gloves
- Must be simple and fast

## Features
1. Parse log files (CSV and TXT formats)
2. Identify critical errors:
   - High coolant temperature
   - Low oil pressure
   - Unusual vibration readings
3. Generate daily summary report
4. Highlight entries that need immediate attention

## Constraints
- Must work offline
- Simple CLI interface (no complex GUI)
- Fast execution (< 5 seconds for 1000 log entries)
- Clear error messages
- No external dependencies beyond Python standard library
```

Run all checks:
```bash
black src/ tests/
ruff check src/ tests/
mypy src/
pytest tests/
```

## Adding New Agents

To add a new agent:

1. Create a new file in `src/code_factory/agents/`
2. Inherit from `BaseAgent`
3. Implement required methods
4. Add tests in `tests/unit/`
5. Update documentation

Example:
```python
from code_factory.core.agent_runtime import BaseAgent

class MyAgent(BaseAgent):
    @property
    def name(self) -> str:
        return "my_agent"

    @property
    def description(self) -> str:
        return "Does something useful"

    def execute(self, input_data: BaseModel) -> BaseModel:
        # Implementation
        pass
```
